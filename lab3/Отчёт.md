### 1. Value Objects (Ценностные Объекты)

**Созданные Value Objects:**

1. **AnnouncementStatus** - статус объявления (DRAFT, SCHEDULED, PUBLISHED, ARCHIVED)
   - Валидация: значение должно быть из Enum
   - Иммутабельность: ✅ (Enum)
   - Файл: `domain/value_objects/announcement_status.py`

2. **FileAttachment** - информация о прикреплённом файле
   - Валидация: проверка file_id, filename, file_size (<=10MB), mime_type
   - Иммутабельность: ✅ (frozen dataclass)
   - Файл: `domain/value_objects/file_attachment.py`

3. **Email** - email пользователя
   - Валидация: проверка формата через regexp
   - Иммутабельность: ✅ (frozen dataclass)
   - Файл: `domain/value_objects/email.py`

4. **AnnouncementId** - идентификатор объявления
   - Валидация: не пустой, не длиннее 50 символов
   - Иммутабельность: ✅ (frozen dataclass)
   - Файл: `domain/value_objects/announcement_id.py`

**Пример кода** (один Value Object):
```python
@dataclass(frozen=True)
class FileAttachment:
    """Value Object: Вложение файла"""
    
    file_id: str
    filename: str
    file_size: int  # bytes
    mime_type: str
    storage_path: str
    
    def __post_init__(self):
        if not self.file_id or not self.file_id.strip():
            raise DomainException("file_id не может быть пустым")
        
        if self.file_size <= 0:
            raise DomainException(f"file_size должен быть положительным: {self.file_size}")
        
        max_size = 10 * 1024 * 1024
        if self.file_size > max_size:
            raise DomainException(f"Файл слишком большой: {self.file_size} байт (макс {max_size})")
```

**Пример кода** (Group):
```python
class Group:
    """Entity: Группа пользователей"""
    
    MIN_MEMBERS = 1
    MAX_MEMBERS = 100
    
    def add_member(self, user_id: str) -> None:
        """Добавить участника"""
        if user_id in self._member_ids:
            raise DomainException(f"Пользователь {user_id} уже в группе")
        
        if len(self._member_ids) >= self.MAX_MEMBERS:
            raise DomainException(f"Группа достигла максимума участников ({self.MAX_MEMBERS})")
        
        self._member_ids.append(user_id)
    
    def add_admin(self, user_id: str) -> None:
        """Назначить администратора"""
        if user_id not in self._member_ids:
            raise DomainException(f"Пользователь {user_id} не состоит в группе")
        
        if user_id not in self._admin_ids:
            self._admin_ids.append(user_id)
```

### 3. Aggregate Root (Корневой агрегат)

**Aggregate Root:** Announcement (Объявление)

**Границы агрегата:**
- Корень: `Announcement`
- Внутренние сущности: `нет (вложения - Value Objects)`
- Value Objects: `AnnouncementStatus, FileAttachment`

**Инварианты агрегата:**

| № | Инвариант | Как проверяется |
|---|----------|----------------|
| 1 | Заголовок не может быть пустым | В _set_title() при создании и обновлении |
| 2 | Заголовок не длиннее 200 символов | len(title) <= MAX_TITLE_LENGTH |
| 3 | Содержимое не может быть пустым | В _set_content() при создании и обновлении |
| 4 |	Содержимое не длиннее 5000 символов |	len(content) <= MAX_CONTENT_LENGTH |
| 5 |	Нельзя опубликовать уже опубликованное объявление |	if status == PUBLISHED в publish() |
| 6 |	Нельзя опубликовать архивное объявление |	if status == ARCHIVED в publish() |
| 7 |	Время публикации при планировании должно быть в будущем |	if scheduled_time <= datetime.now() |
| 8 |	Максимум 5 файлов на объявление |	if len(attachments) >= MAX_ATTACHMENTS |
| 9	| Нельзя изменить содержимое не в черновике |	if status != DRAFT в update_content() |

**Пример кода Aggregate Root:**
```python
class Announcement:
    """Aggregate Root: Объявление в группе"""
    
    MAX_ATTACHMENTS = 5
    MAX_TITLE_LENGTH = 200
    MAX_CONTENT_LENGTH = 5000
    
    def publish(self) -> None:
        """Опубликовать объявление"""
        if self._status == AnnouncementStatus.PUBLISHED:
            raise DomainException("Объявление уже опубликовано")
        
        if self._status == AnnouncementStatus.ARCHIVED:
            raise DomainException("Нельзя опубликовать архивное объявление")
        
        if self._status == AnnouncementStatus.SCHEDULED:
            if self._scheduled_for and self._scheduled_for > datetime.now():
                raise DomainException(f"Время публикации ещё не наступило")
        
        self._status = AnnouncementStatus.PUBLISHED
        self._published_at = datetime.now()
        
        self._register_event(AnnouncementPublishedEvent(
            announcement_id=self._id,
            group_id=self._group_id,
            published_at=self._published_at
        ))
    
    def add_attachment(self, attachment: FileAttachment) -> None:
        """Добавить вложение"""
        if self._status != AnnouncementStatus.DRAFT:
            raise DomainException(f"Нельзя добавить вложение в статусе {self._status}")
        
        if len(self._attachments) >= self.MAX_ATTACHMENTS:
            raise DomainException(f"Максимум {self.MAX_ATTACHMENTS} файлов на объявление")
        
        if any(a.file_id == attachment.file_id for a in self._attachments):
            raise DomainException(f"Файл {attachment.file_id} уже прикреплён")
        
        self._attachments.append(attachment)
```

### 4. Domain Events (Доменные события)

**Созданные события:**

1. **AnnouncementPublishedEvent** - генерируется при публикации объявления
   - Данные: `announcement_id, group_id, published_at`
   - Файл: `domain/events/announcement_events.py`

2. **AnnouncementScheduledEvent** - генерируется при планировании публикации
   - Данные: `announcement_id, scheduled_for`
   - Файл: `domain/events/announcement_events.py`

3. **AnnouncementArchivedEvent** - генерируется при архивации
   - Данные: `announcement_id`
   - Файл: `domain/events/announcement_events.py`

4. **AttachmentAddedEvent** - генерируется при добавлении вложения
   - Данные: `announcement_id, file_id, filename`
   - Файл: `domain/events/announcement_events.py`

5. **AttachmentRemovedEvent** - генерируется при удалении вложения
   - Данные: `announcement_id, file_id`
   - Файл: `domain/events/announcement_events.py`

**Пример кода события:**
```python
class AnnouncementPublishedEvent(DomainEvent):
    """Событие: объявление опубликовано"""
    
    def __init__(self, announcement_id: str, group_id: str, published_at: datetime):
        self._announcement_id = announcement_id
        self._group_id = group_id
        self._occurred_at = published_at
    
    @property
    def announcement_id(self) -> str:
        return self._announcement_id
    
    def occurred_at(self) -> datetime:
        return self._occurred_at
    
    def get_name(self) -> str:
        return "announcement.published"
```
